# CollabHub 

CollabHub is a comprehensive team collaboration and task management platform. 

## Key Modules

- **Authentication & Profiles:** Secure user login, registration, and personalized user profiles.
- **Team Management:** Create teams, manage members, and assign role-based access (like Team Leaders).
- **Dashboards:** Role-specific dashboards (e.g., Team Leader Dashboard) for an overview of progress and activities.
- **Task & Subtask Management:** Hierarchical task tracking allowing users to create main tasks and break them down into manageable subtasks.

## Purpose

The frontend is built to provide an intuitive interface for users to coordinate on projects, track task completion statuses, and manage team workflows efficiently.
